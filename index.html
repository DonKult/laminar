Simplicity

Everywhere you are

Laminar tracks your automated jobs. And that's all it does. No clumsy configuration GUIs, no source-control or build system lock-in.

Laminar's web frontend is a simple and clean bootstrap-based design that looks great on all devices.

Laminar's backend is a super-lightweight systemd service written in C++ that leverages the underlying Linux system rather than reinventing the wheel. 

Full Control

If you need continuous integration, you  don't need anything getting in the way between you and your config files. Laminar supports all external services equally - by forking to your bash scripts and passing environment variables. This keeps you in control without introducing bloat. 

See the wiki page for tutorials and example usage patterns.

Self-Hosted and Open Source

Stay in control of your build infrastructure and freely  customize Laminar to suit your needs.
